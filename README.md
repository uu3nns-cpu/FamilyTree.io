## 📋 프로젝트 소개

가족관계도(Genogram)는 상담, 사회복지, 사례관리 등에서 자주 활용되는 도구이지만, 기존 방식으로 작성하려면 시간이 오래 걸리고 사용 과정이 복잡한 경우가 많습니다. 특히 무료 도구는 기능이 제한적이거나 수정에 불편함이 있고, 반대로 유료 프로그램은 필요한 기능보다 더 많은 요소를 포함하고 있어 부담이 될 수 있습니다.

**결속**은 이러한 불편함을 줄이기 위해 제작되었습니다.

### 🎯 핵심 목표

- **필요한 기능만 간단하게 제공**
- **설치나 회원가입 없이 바로 사용 가능**
- **작성과 수정이 빠르고 반복 작업 최소화**

복잡한 설정이나 학습이 필요한 구조가 아니라, 기본적인 가족관계도 제작 흐름에 맞춰 설계되었습니다. 생성, 관계 연결, 구성 수정 등의 기능을 사용자가 직관적으로 사용할 수 있도록 화면 구성과 버튼 구조가 간단하게 배치되어 있습니다.

특히 상담 현장에서 반복적으로 작성해야 하는 가족 구성도나 기본 사례 템플릿을 빠르게 구성할 수 있도록 정렬과 관계 배치가 자동화되어 있어, 작성자가 형태나 비례를 직접 조정할 필요가 없습니다. 이를 통해 시간이 많이 소요되던 초기 구성 작업이 줄어듭니다.

> **이 프로젝트의 목적은 복잡한 기능을 계속 추가하는 것이 아니라,  
> 실제 현장에서 자주 쓰이는 방식에 맞추어 효율을 높이는 것입니다.**

가족관계도를 그리는 과정의 난이도를 높이는 대신, 필요한 요소만 빠르게 채울 수 있도록 구성한 점이 가장 큰 특징입니다.

이 도구가 상담 현장이나 교육 환경에서 반복적인 문서 작성 부담을 줄이고, 보다 간편하게 가족구조 이해 작업을 수행하는 데 도움이 되기를 기대합니다.

## ❓ 자주 묻는 질문 (FAQ)

### Q. 이 도구는 어떤 목적을 가지고 만들어졌나요?

가족관계도를 자주 작성해야 하는 상담사·사회복지사·교육자들이 불필요하게 복잡한 프로그램을 사용하거나 동일한 작업을 반복하는 문제를 줄이기 위해 개발되었습니다. **"빠르게 만들고, 쉽게 수정하는 것"**이 핵심 목표입니다.

### Q. 사용법이 어려운가요? 따로 학습이 필요한가요?

별도의 매뉴얼 없이도 사용할 수 있도록 설계되었습니다. 가족 구성원을 추가하고, 관계를 연결하면 자동으로 정렬되며 버튼과 레이아웃이 최소화되어 있어 처음 사용하는 사람도 쉽게 적응할 수 있습니다.

### Q. 기존 무료 도구와 무엇이 다른가요?

일부 무료 도구는 기능이 제한적이거나, 반대로 너무 많은 옵션이 포함되어 난이도가 높습니다. 이 도구는 실제로 자주 쓰는 기능만 제공하며, 불필요한 설정과 옵션을 최소화했습니다.

## 🚀 주요 기능

### 📊 템플릿 & 빠른 시작
- **빈 캔버스**: 완전히 새로운 구조로 자유롭게 시작
- **1가족 3세대**: 조부모-부모-자녀 구조의 표준 템플릿 (권장)
- **2가족 3세대**: 재혼 가정이나 복잡한 가족 구조용
- **3세대 확장**: 4세대까지 확장 가능
- 인물, 부부, 부모-자식 관계 자동 생성
- 클라이언트(CT) 중심 자동 배치

### 👥 인물 관리
- **빠른 추가**: 우클릭 메뉴로 부모/자녀/형제자매/배우자 즉시 추가
- **속성 편집**: 인물 클릭 시 우측 패널에서 이름, 나이, 성별, 생존 상태, 메모 편집
- **다중 선택**: Ctrl+클릭, Shift+드래그, Ctrl+A로 다중 선택 후 일괄 이동/삭제

### 💞 관계 표현
- **부부 관계**: 결혼/동거/재혼/별거/이혼 등 다양한 관계 타입
- **부모-자식 관계**: 자동으로 형제자매 연결
- **감정선 (10가지)**: 조화, 친밀, 사랑, 거리감, 단절, 불화, 적대, 융합, 학대, 조종
- **감정선 편집**: 감정선 클릭 시 우측 패널에서 유형 변경 및 메모 추가

### 🎨 시각화
- **레이어 토글**: 이름/나이/관계선/감정선/범례 개별 표시/숨김
- **선 굵기 조정**: 인물 테두리, 관계선, 감정선 굵기 독립 조절
- **감정선 투명도**: 복잡한 관계도에서 가독성 향상
- **그리드 모드**: 실선/점선/숨김 3가지 옵션
- **자석 정렬**: 노드 이동 시 그리드에 자동 정렬

### 📐 레이아웃
- **자동 정렬**: 표준 가계도 형식으로 한 번에 정리
- **수동 조정**: 드래그로 자유롭게 위치 조정
- **다중 이동**: 여러 노드 선택 후 일괄 이동
- **줌 & 팬**: 마우스 휠 확대/축소, 화살표 키 이동

### 💾 저장 & 내보내기
- **자동 저장**: 작업 중 주기적으로 자동 저장
- **수동 저장**: Ctrl+S 단축키 또는 저장 버튼
- **프로젝트 관리**: 최근 프로젝트 목록, 프로젝트 이름 변경
- **다양한 형식**:
  - JSON: 프로젝트 데이터 저장 및 불러오기
  - PNG: 이미지로 내보내기 (프레젠테이션, 보고서용)
  - SVG: 벡터 이미지로 내보내기 (확대 가능, 인쇄용)

### ⚙️ 설정 & 커스터마이징
- **테마**: 라이트/다크 모드 지원
- **UI 배율**: 100%~150% 조정 가능
- **튜토리얼**: 처음 사용자를 위한 단계별 가이드
- **설정 초기화**: 모든 설정을 기본값으로 되돌리기

### ⌨️ 생산성 기능
- **실행 취소/다시 실행**: Ctrl+Z/Ctrl+Y (최대 50단계)
- **키보드 단축키**: Ctrl+S(저장), Delete(삭제), Ctrl+A(전체 선택) 등
- **컨텍스트 메뉴**: 우클릭으로 모든 주요 기능 접근
- **툴팁**: 모든 버튼에 설명 표시

- `Ctrl + 클릭` : 다중 선택
