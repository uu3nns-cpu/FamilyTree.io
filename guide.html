1

화면 구성

캔버스 화면은 세 영역으로 나뉩니다. 상단 툴바에는 자동정렬·간격 조절·실행 취소·저장·내보내기·설정 버튼이 있고, 중앙 캔버스가 실제 작업 공간입니다. 우측 하단의 ＋ / － 버튼으로 확대·축소하며, 비율 숫자를 클릭하면 100%로 돌아옵니다.

💡 캔버스에서 일어나는 대부분의 작업은 인물을 우클릭하는 데서 시작합니다.
2

인물 도형 읽는 법

도형의 모양과 표시로 성별·상태를 나타냅니다.

사각형남성
원여성
다이아몬드성별 미상

성별과 상태는 인물을 더블클릭해 열리는 편집창에서 지정합니다.

3

인물 선택 및 이동

  • 클릭 — 인물 선택 (파란 테두리)
  • Ctrl + 클릭 — 복수 선택 / 선택 해제
  • 빈 캔버스 드래그 — 범위 안 인물 한꺼번에 선택
  • 선택 후 드래그 — 이동. 설정의 '자석 모드'를 켜면 격자에 맞춰 정렬됩니다
4

인물 정보 편집

전체 편집

인물을 더블클릭하면 이름·성별·나이·CT 여부·사망 여부를 한 번에 수정할 수 있습니다. '저장'을 누르면 캔버스에 바로 반영됩니다.

빠른 편집

이름이나 나이만 바꾸려면 우클릭 → 편집 → 이름 / 나이를 선택하세요. 캔버스 위에 작은 입력창이 나타나며, Enter로 저장하거나 Esc로 취소합니다.

CT·사망 표시는 우클릭 → 상태에서 클릭할 때마다 켜고 끌 수 있습니다.

5

가족관계로 인물 추가

인물을 우클릭한 뒤 원하는 관계를 선택하면 새 인물이 자동으로 생성되고 관계선도 함께 그려집니다.

  • 부모 추가 — 아버지+어머니 / 아버지만 / 어머니만. 이미 같은 부모가 있으면 추가되지 않습니다
  • 배우자 추가 — 기존 배우자가 있으면 오른쪽에 새 배우자를 추가하고 '재혼'으로 표시됩니다
  • 자녀 추가 — 아들 / 딸 / 성별 미상. 배우자가 있으면 배우자도 자동으로 부모로 연결됩니다
  • 형제자매 추가 — 부모가 먼저 연결되어 있어야 추가할 수 있습니다
💡 주요인물(CT)을 기준으로 세대를 계산해 이름을 자동으로 지어줍니다. CT의 부모에게 '부모 추가'를 하면 할아버지·할머니, 그 위 세대는 증조부모·고조부모로 명명됩니다.
6

감정선 연결

인물 사이의 감정적 분위기를 선으로 표시할 수 있습니다. 지원 유형은 다음과 같습니다.

긍정
친밀한 관계, 사랑
거리·단절
거리감, 단절
갈등
갈등, 적대적 관계
복합
융합
학대·방임
신체적 학대, 정서적 학대, 성적 학대, 방임

연결 방법

  1. 첫 번째 인물을 우클릭 → 감정선 연결에서 유형을 선택합니다
  2. 도형에 초록색 점선 테두리가 나타납니다
  3. 두 번째 인물을 클릭하면 감정선이 그려집니다
📌 이미 감정선이 있는 두 사람을 같은 방법으로 다시 연결하면 기존 선이 새 유형으로 교체됩니다. 굵기·투명도·표시 여부는 ⚙️ 설정에서 조정할 수 있습니다.
7

인물·관계 삭제

  • 인물 우클릭 → 삭제
  • 인물 선택 후 Delete 또는 Backspace — 여러 명 동시 삭제 가능
  • 관계선·감정선은 우클릭 → 삭제로만 지울 수 있습니다
⚠️ 인물을 삭제하면 연결된 모든 관계선·감정선도 함께 삭제됩니다. 실수로 지웠다면 실행 취소로 복구하세요.
8

캔버스 보기 조작

  • 마우스 휠 — 확대·축소
  • Space + 드래그 — 화면 이동(팬)
  • 자동정렬 버튼 (또는 Ctrl+Shift+L) — 세대·관계에 맞춰 한 번에 재배치
  • 툴바의 세로간격 / 가로간격 ＋ / － — 인물 간 간격 조절 (인물 2명 이상일 때 동작)
💡 인물을 여러 명 추가한 뒤 자동정렬을 한 번 눌러주면 가계도가 훨씬 깔끔해집니다.
9

설정

툴바 우측의 ⚙️ 설정을 클릭하면 설정창이 열립니다.

  • 표시 옵션 — 이름·나이·사망·감정선 표시 토글, 그리드 스타일(없음·점선·실선)
  • 자동 저장 — 켜고 끄기, 저장 간격(10초·30초·1분·5분)
  • 캔버스 — 기본 선 두께, 감정선 두께·투명도 조절, 자석 모드

마음에 들지 않으면 '기본값으로 재설정'을 눌러 한 번에 되돌릴 수 있습니다.

10

저장 · 불러오기 · 내보내기

일시저장 / 불러오기

툴바의 '일시저장'을 누르고 이름을 입력하면 저장됩니다. 프로젝트는 최대 9개까지 저장할 수 있으며, '불러오기'에서 목록을 확인하고 전환하거나 삭제할 수 있습니다.

⚠️ 다른 프로젝트를 불러오면 현재 작업 내용은 저장되지 않습니다. 불러오기 전에 먼저 저장하세요.

내보내기

툴바의 '내보내기'를 누르면 미리보기와 함께 파일로 저장할 수 있습니다.

PNG고해상도 이미지. 문서 삽입·일반 인쇄에 적합합니다
SVG벡터 이미지. 확대해도 깨지지 않아 대형 출력에 적합합니다
11

실행 취소 · 다시 실행

툴바의 '실행 취소' / '다시 실행'으로 최근 50단계까지 되돌리거나 다시 적용할 수 있습니다. 인물을 잘못 삭제하거나 자동정렬 결과가 마음에 들지 않을 때 바로 눌러보세요.

12

자주 묻는 상황

인물이 겹쳐 보여요

자동정렬 버튼을 눌러보세요.

인물이 화면 밖으로 사라졌어요

휠로 축소하거나 Space+드래그로 화면을 옮겨보세요.

가계도가 너무 빽빽해요

세로·가로 간격 ＋ 버튼으로 여유를 줘보세요.

형제자매가 따로 떨어져 보여요

부모가 먼저 연결되어 있는지 확인하세요.

인쇄하거나 문서에 넣고 싶어요

내보내기에서 PNG로 저장한 뒤 삽입하세요.

감정선이 너무 복잡해요

설정에서 '감정선 표시'를 잠시 꺼두세요.
