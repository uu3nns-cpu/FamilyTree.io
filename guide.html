영역 위치 할 수 있는 일
상단 툴바 화면 맨 위 자동정렬, 간격 조절, 실행 취소·다시 실행, 일시저장, 불러오기, 내보내기

확대/축소 컨트롤 화면 우측 하단 ＋ / 현재 비율(클릭 시 100%로) / －
헤더 화면 맨 위, 브랜드 좌우 좌측: 테마 전환 · 우측: ⚙️ 설정
💡 가장 많이 쓰는 동작 한 가지만 기억하세요: 캔버스에서 일어나는 대부분의 작업(인물 추가, 편집, 감정선 연결, 삭제)은 인물을 마우스 오른쪽 클릭하는 데서 시작합니다.

2️⃣ 인물 선택하고 이동하기

  • 클릭 — 인물 한 명 선택 (파란 테두리로 표시)
  • Ctrl+클릭 (Mac: Cmd+클릭) — 선택을 유지한 채 다른 인물을 추가하거나 선택에서 제외
  • 빈 캔버스를 드래그 — 영역 안에 있는 인물을 한꺼번에 선택
  • 선택된 인물을 드래그 — 함께 이동. ⚙️ 설정의 '자석 모드'를 켜두면 격자에 맞춰 정렬됩니다
  • 캔버스 빈 곳 우클릭 → '전체 선택' / '선택 해제'

3️⃣ 인물 정보 편집하기

전체 정보 편집 (더블클릭)

인물을 더블클릭하면 편집창이 열립니다. 다음 항목을 한 번에 수정할 수 있어요:

  • 이름 (필수)
  • 성별 — 남성 / 여성 / 미상
  • 나이 — 0~150 사이 숫자
  • 상태 — 주요인물(CT), 사망 여부 체크박스

입력 후 '저장' 버튼을 누르면 바로 캔버스에 반영됩니다.

빠른 편집 (우클릭)

이름이나 나이만 빠르게 바꾸고 싶다면 인물을 우클릭 → 편집 → 이름 / 나이를 선택하세요. 캔버스 위에 작은 입력창이 바로 나타나며, Enter로 저장하거나 Esc로 취소할 수 있습니다(입력창 밖을 클릭해도 저장됩니다).

주요인물(CT)·사망 표시는 우클릭 → 상태 → CT(주요인물) / 사망을 클릭할 때마다 켜고 끌 수 있습니다.

4️⃣ 가족관계로 인물 추가하기

인물을 우클릭한 뒤 원하는 관계를 선택하면, 새 인물이 자동으로 만들어지고 관계선도 함께 그려집니다.

👨‍👩‍👧 부모 추가

아버지+어머니 / 아버지만 / 어머니만. 이미 같은 부모가 있으면 추가되지 않고 안내 메시지가 표시됩니다.

💑 배우자 추가

이미 배우자가 있으면 오른쪽에 새 배우자를 추가하고 '재혼'으로 표시됩니다.

👶 자녀 추가

아들 / 딸 / 성별미상. 배우자가 있다면 배우자도 자동으로 부모로 연결됩니다.

👫 형제자매 추가

남자형제 / 여자형제 / 성별미상. 부모가 먼저 연결되어 있어야 추가할 수 있습니다.

💡 이름이 자동으로 붙어요: 주요인물(CT)을 기준으로 세대를 계산해 이름을 지어줍니다. 예를 들어 CT의 부모에게 '부모 추가'를 하면 할아버지/할머니(또는 외할아버지/외할머니), 그 위 세대는 증조부모·고조부모로 자동 명명됩니다.
⚠️ 주의: 형제자매는 부모가 없으면 추가되지 않아요. 먼저 '부모 추가'로 부모를 연결한 뒤 형제자매를 추가해주세요.

5️⃣ 감정선 연결하기

가족 관계뿐 아니라 인물 사이의 감정적인 분위기도 표시할 수 있습니다. 지원하는 11가지 유형은 다음과 같습니다.

분류 유형
긍정적 관계 친밀한 관계, 사랑
거리·단절 거리감, 단절
갈등 갈등, 적대적 관계
복합 융합
학대·방임 신체적 학대, 정서적 학대, 성적 학대, 방임

연결하는 방법

  1. 첫 번째 인물을 우클릭 → '감정선 연결' 서브메뉴에서 원하는 유형을 선택하세요
  2. 선택한 인물 도형에 초록색 점선 테두리가 표시됩니다
  3. 이어서 연결할 두 번째 인물을 클릭하면 두 사람 사이에 감정선이 그려집니다
📌 다시 연결하면 바뀌어요: 같은 방향으로 이미 감정선이 있는 두 사람을 같은 방법으로 다시 연결하면 기존 감정선이 새 유형으로 바뀝니다. 반대 방향의 감정선이 있다면 기존 선은 제거되고 새 방향·유형으로 다시 그려집니다.

감정선의 굵기·투명도·표시 여부는 ⚙️ 설정에서 조정할 수 있습니다 (다음 항목 참고).

6️⃣ 인물·관계 삭제하기

  • 인물 우클릭 → 삭제 (확인창이 한 번 더 나타납니다)
  • 인물을 선택한 뒤 Delete 또는 Backspace — 선택된 인물을 한 번에 모두 삭제 (여러 명 동시 가능)
  • 관계선·감정선은 우클릭 → 삭제로만 지울 수 있습니다 (키보드로는 삭제되지 않습니다)
⚠️ 주의: 인물을 삭제하면 그 인물과 연결된 모든 관계선·감정선도 함께 삭제됩니다. 실수로 지웠다면 '실행 취소'로 복구할 수 있습니다.

7️⃣ 캔버스 보기 조작하기

  • 마우스 휠 — 확대/축소
  • 우측 하단 ＋ / － 버튼 — 확대/축소, 가운데 비율을 클릭하면 100%로 초기화
  • Space를 누른 채 드래그 — 화면 이동(팬)
  • 툴바의 '자동정렬' 버튼 (또는 Ctrl+Shift+L) — 인물을 세대와 관계에 맞춰 한 번에 재배치
  • 툴바의 세로간격 / 가로간격 ＋ / － 버튼 — 인물 사이 간격을 한 단계씩 넓히거나 좁힘 (인물이 2명 이상일 때 동작)
💡 팁: 인물을 여러 명 추가한 뒤에는 자동정렬을 한 번 눌러주면 가계도가 훨씬 깔끔해집니다.

8️⃣ 설정(⚙️)에서 화면 조정하기

헤더 우측의 ⚙️ 아이콘을 클릭하면 설정창이 열립니다.

표시 옵션

  • 이름 표시 / 나이 표시 / 사망 표시 / 감정선 표시 — 각각 켜고 끄는 토글
  • 그리드 표시 — 없음 / 점선 / 실선

자동 저장

  • 자동 저장 켜고 끄기
  • 저장 간격 — 10초 / 30초 / 1분 / 5분

캔버스 설정

  • 기본 선 두께(도형·기본 관계선), 감정선 두께, 감정선 투명도 — 각각 −/기본/＋ 버튼으로 조정
  • 자석 모드 — 켜두면 인물을 그리드에 맞춰 이동시킵니다

모두 바꿔보고 마음에 들지 않으면 '기본값으로 재설정' 버튼으로 한 번에 되돌릴 수 있습니다(재설정 후 페이지가 새로고침됩니다).

9️⃣ 저장 · 불러오기 · 내보내기

일시저장

툴바의 '일시저장'을 클릭하고 프로젝트 이름을 입력하면 저장됩니다. 프로젝트는 최대 9개까지 저장할 수 있습니다.

불러오기

툴바의 '불러오기'를 클릭하면 저장된 프로젝트 목록이 나타납니다. 목록에서 다른 프로젝트를 선택해 불러오거나 🗑️ 아이콘으로 삭제할 수 있습니다.

⚠️ 주의: 다른 프로젝트를 불러오면 현재 작업 중인 내용은 저장되지 않습니다. 불러오기 전에 먼저 일시저장을 해두는 것이 안전합니다.

내보내기

툴바의 '내보내기'를 클릭하면 가계도와(감정선이 있다면) 감정선 기호 설명을 미리보기로 함께 확인할 수 있습니다. 파일명을 입력하고 형식을 고른 뒤 각각 따로 다운로드합니다.

형식 특징 권장 용도
PNG 고해상도 비트맵 이미지 문서 삽입, 일반 인쇄, 웹 공유
SVG 벡터 이미지 (확대해도 깨지지 않음) 정밀한 인쇄물, 대형 출력

미리보기 영역의 '크게 보기'를 누르면 별도 팝업창에서 전체 크기로 확인할 수 있습니다.

🔟 실행 취소 · 다시 실행

툴바의 '실행 취소' / '다시 실행' 버튼으로 최근 50단계까지 되돌리거나 다시 적용할 수 있습니다.

💡 팁: 인물을 잘못 삭제했거나 자동정렬 결과가 마음에 들지 않을 때 바로 '실행 취소'를 눌러보세요.

💡 자주 묻는 상황

  • 인물이 겹쳐 보여요 — 자동정렬 버튼을 눌러보세요
  • 인물이 화면 밖으로 사라졌어요 — 마우스 휠로 축소하거나 Space+드래그로 화면을 옮겨보세요
  • 가계도가 너무 빽빽해요 — 세로/가로 간격 ＋ 버튼으로 여유를 줘보세요
  • 형제자매를 추가했는데 따로 떨어져 보여요 — 부모가 먼저 연결되어 있는지 확인해보세요
  • 인쇄하거나 문서에 넣고 싶어요 — 내보내기에서 PNG로 저장한 뒤 문서에 삽입하세요
  • 감정선이 너무 많아 복잡해요 — 설정에서 '감정선 표시'를 잠시 꺼두고 가족 구조만 확인해보세요
