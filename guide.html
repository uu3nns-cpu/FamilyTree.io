📖 사용 안내서

결속(Gyeolsok)으로 가계도를 만드는 방법을 단계별로 안내합니다

🚀 5단계로 시작하기

  1. 템플릿 선택 - 좌측 패널에서 적절한 템플릿 클릭
  2. 인물 선택 - 수정하려는 인물 클릭
  3. 속성 편집 - 우측 패널에서 이름, 나이, 성별 등 수정
  4. 인물 추가 - 우클릭 메뉴로 부모, 자녀, 배우자 추가
  5. 저장 - Ctrl+S로 프로젝트 저장 또는 내보내기

1️⃣ 템플릿 선택하기

캔버스 좌측 패널의 "템플릿" 섹션에서 가족 구조에 맞는 템플릿을 선택하세요. 템플릿을 클릭하면 즉시 캔버스에 로드됩니다.

사용 가능한 6가지 템플릿

템플릿명 구성 권장 사용 상황
튜토리얼 시작 내담자 1명 빈 캔버스에서 처음부터 만들 때
2세대 1가정 미니멀 부모 + 내담자 (4명) 간단한 핵가족
2세대 1가정 형제많음 부모 + 내담자 + 형제 3명 (6명) 형제가 많은 가족
3세대 1가정 표준 조부모 + 부모 + 내담자 + 배우자 + 자녀 (7명) 가장 일반적인 가계도 (⭐ 권장)
3세대 2가정 친가·외가 조부모 + 부모 + 내담자 (7명) 양가 가족 관계 분석
4세대 2가정 최대확장 증조부모 + 조부모 + 부모 + 내담자 (11명) 복잡한 대가족 구조
💡 팁: 처음 사용하신다면 3세대 1가정 표준 템플릿을 추천합니다. 가장 일반적인 구조로 쉽게 시작할 수 있습니다.

2️⃣ 인물 편집 및 추가

인물 속성 편집하기

수정하려는 인물을 클릭하면 우측 패널에 속성 편집 패널이 나타납니다. 여기에서 모든 정보를 수정할 수 있습니다:

  • 이름 - 인물의 이름 입력 및 수정
  • 나이 - 나이 입력 (숫자만)
  • 성별 - 남성 / 여성 / 성별미상 선택
  • 생존 여부 - 생존 / 사망 선택 (사망 시 X 표시)
  • 메모 - 추가 정보나 특이사항 기록
📌 편집 방법: 인물 노드를 클릭하면 우측 속성 패널이 자동으로 열립니다. 원하는 항목을 수정한 후 다른 곳을 클릭하면 자동으로 저장됩니다.

우클릭 메뉴로 인물 추가

인물 노드를 우클릭하면 컨텍스트 메뉴가 나타납니다. 다음 옵션을 선택할 수 있습니다:

👨 부모 추가

아버지 또는 어머니 추가. 자동으로 부모-자녀 관계선 생성

💑 배우자 추가

배우자 추가. 자동으로 부부 관계선 생성

👶 자녀 추가

아들, 딸, 성별미상 자녀 선택 가능

👫 형제자매 추가

남자, 여자, 성별미상 형제자매 선택

💝 감정선 연결

다른 인물과 감정 관계 설정

🗑️ 삭제

인물 및 관련 관계선 모두 삭제

여러 인물 동시에 선택하기

다수의 인물을 한 번에 선택하여 작업할 수 있습니다:

  • Ctrl+클릭 (Mac: Cmd+클릭) - 개별 인물을 추가로 선택하거나 선택 해제
  • 드래그 선택 - 마우스로 영역을 드래그하여 여러 인물 한 번에 선택
  • Ctrl+A (Mac: Cmd+A) - 캔버스의 모든 인물 선택
  • 일괄 이동 - 선택된 인물들을 드래그하여 함께 이동
  • Delete - 선택된 모든 인물 일괄 삭제
📌 참고: 인물을 삭제하면 해당 인물과 연결된 모든 관계선과 감정선도 함께 삭제됩니다.

3️⃣ 관계 연결하기

가족 관계선 (자동 생성)

우클릭 메뉴로 부모, 자녀, 배우자를 추가하면 자동으로 관계선이 생성됩니다:

  • 부모-자식 관계 - 수직선으로 표현
  • 부부 관계 - 수평선으로 표현
  • 형제자매 관계 - 같은 부모의 자녀들을 연결

감정선 연결하기

좌측 패널의 "감정선" 섹션에서 원하는 감정 유형을 선택한 후, 연결할 두 인물을 순서대로 클릭하면 감정선이 생성됩니다.

지원하는 10가지 감정선

카테고리 감정선 유형 설명
긍정 조화 평온하고 안정적인 관계
친밀 가깝고 친근한 관계
사랑 강한 애정과 유대감
거리감 거리감 멀어지고 소원한 관계
단절 완전히 끊긴 관계
갈등 불화 다툼과 갈등이 있는 관계
적대 극심한 대립과 반목
특수 융합 경계가 모호한 밀착 관계
학대 폭력이나 학대 관계
조종 통제와 조작이 있는 관계

감정선 수정 및 삭제

생성된 감정선을 클릭하면 우측 패널에서 다음 작업이 가능합니다:

  • 감정선 유형 변경 - 드롭다운에서 다른 유형 선택
  • 메모 추가 - 관계에 대한 상세 설명 입력
  • 삭제 - 감정선 제거 버튼 클릭
📌 편집 방법: 감정선을 클릭하면 우측 속성 패널이 열리며, 감정선의 유형과 메모를 수정할 수 있습니다.

감정선 범례 내보내기

좌측 패널 하단의 "범례 내보내기" 버튼을 클릭하면 감정선 범례를 PNG 또는 SVG 파일로 저장할 수 있습니다. 보고서나 문서에 참고용으로 첨부하기 좋습니다.

4️⃣ 캔버스 조작하기

확대/축소 (줌)

  • 마우스 휠 - 위로 굴리면 확대, 아래로 굴리면 축소 (10%~300% 범위)
  • 툴바 버튼 - 상단 툴바의 + / - 버튼 클릭
  • 줌 리셋 - 100% 버튼으로 원래 크기로 복원

캔버스 이동 (팬)

  • Space를 누른 상태에서 마우스 드래그 - 캔버스 전체를 이동
  • 줌 인 상태에서 원하는 영역을 빠르게 찾을 때 유용

자동정렬 기능

툴바의 "자동정렬" 버튼을 클릭하면 모든 인물이 세대별로 자동 정렬됩니다:

  • 세대별로 수평 정렬
  • 부모-자식 관계를 고려한 수직 배치
  • 겹치는 요소 자동 조정
  • 관계선이 최대한 직선으로 표시되도록 최적화
💡 팁: 인물을 여러 개 추가한 후에는 자동정렬 기능을 사용하면 가계도가 훨씬 깔끔하게 정리됩니다.

그리드 및 마그넷 설정

툴바의 설정 버튼 (⚙️)을 클릭하면 다음 옵션을 설정할 수 있습니다:

  • 그리드 표시 - 실선 / 점선 / 없음 선택
  • 마그넷 정렬 - 활성화 시 인물이 그리드에 자동으로 정렬됨
  • UI 스케일 - 화면 크기에 따라 인터페이스 배율 조정

5️⃣ 레이어 표시 설정

툴바의 레이어 토글 버튼으로 표시할 요소를 자유롭게 선택할 수 있습니다. 특정 정보만 보거나 내보내기할 때 유용합니다.

토글 가능한 레이어

👤 인물

인물 노드 표시/숨기기

🔗 관계선

가족 관계선 표시/숨기기

💝 감정선

감정선 표시/숨기기

📛 이름

인물 이름 표시/숨기기

🔢 나이

나이 정보 표시/숨기기

📋 범례

감정선 범례 표시/숨기기

📌 중요: 내보내기 시 현재 표시된 레이어만 이미지에 포함됩니다. 원하는 요소만 선택하여 내보내세요.

활용 예시

  • 간단한 가족 구조만 보기 - 감정선 숨김, 관계선만 표시
  • 감정 관계 분석 - 감정선만 표시, 관계선 숨김
  • 깔끔한 프레젠테이션 - 범례와 나이 숨김, 이름과 인물만 표시

6️⃣ 저장 및 내보내기

프로젝트 저장

작업 중인 가계도는 브라우저의 LocalStorage에 자동으로 저장됩니다:

  • Ctrl+S (Mac: Cmd+S) - 수동으로 즉시 저장
  • 자동 저장 - 변경사항이 생기면 자동으로 저장됨
  • 최근 프로젝트 - 최대 10개까지 저장되며, 대시보드에서 확인 가능
  • 프로젝트 이름 - 저장 시 프로젝트 이름 지정 가능
⚠️ 주의: 브라우저 캐시를 지우거나 시크릿 모드를 사용하면 저장된 프로젝트가 삭제될 수 있습니다. 중요한 작업은 이미지로 내보내기를 권장합니다.

이미지로 내보내기

툴바의 "내보내기" 버튼을 클릭하면 다음 형식으로 다운로드할 수 있습니다:

형식 특징 권장 용도
PNG 고해상도 비트맵 (2배 스케일) 문서 삽입, 일반 인쇄, 웹 공유
SVG 벡터 이미지 (무한 확대 가능) 전문 인쇄물, 로고, 대형 출력

내보내기 팁

  • 내보내기 전에 레이어 토글로 필요한 요소만 표시하세요
  • 줌을 100%로 맞추고 내보내면 가장 깔끔합니다
  • 큰 가계도는 SVG 형식이 더 선명합니다

7️⃣ 실행 취소 및 재실행

결속은 모든 작업을 기록하여 언제든 되돌리거나 다시 실행할 수 있습니다:

  • Ctrl+Z (Mac: Cmd+Z) - 실행 취소 (Undo)
  • Ctrl+Y (Mac: Cmd+Y) - 다시 실행 (Redo)
  • 툴바의 ↶ / ↷ 버튼으로도 가능
  • 최대 50단계까지 되돌릴 수 있음
💡 팁: 실수로 인물을 삭제했거나 잘못된 작업을 했다면 즉시 Ctrl+Z로 되돌릴 수 있습니다.

8️⃣ 단축키 모음

자주 사용하는 작업을 빠르게 수행할 수 있는 단축키입니다:

단축키 기능
Ctrl+S / Cmd+S 프로젝트 저장
Ctrl+Z / Cmd+Z 실행 취소 (Undo)
Ctrl+Y / Cmd+Y 다시 실행 (Redo)
Ctrl+A / Cmd+A 전체 선택
Ctrl+클릭 / Cmd+클릭 다중 선택 (개별 추가/제거)
Delete / Backspace 선택 항목 삭제
Esc 선택 해제 / 작업 취소
Space + 드래그 캔버스 이동 (팬)
클릭 인물/감정선 선택 및 속성 편집
우클릭 컨텍스트 메뉴 열기

💡 유용한 팁

⚡ 작업 효율 높이기
  • 작업 중 자주 Ctrl+S로 저장하세요
  • 복잡한 가계도는 템플릿으로 시작하면 시간을 절약할 수 있습니다
  • 다중 선택을 활용하여 여러 인물을 한 번에 이동하세요
  • 자동정렬 기능으로 깔끔하게 정리하세요
  • 레이어 토글로 필요한 정보만 표시하여 집중하세요
  • 인물이나 감정선을 클릭하여 우측 패널에서 속성을 바로 편집하세요
⚠️ 주의사항
  • 브라우저 캐시를 지우면 저장된 프로젝트가 삭제됩니다
  • 시크릿/사생활 보호 모드에서는 프로젝트가 저장되지 않습니다
  • 중요한 작업은 PNG나 SVG로 백업하세요
  • 인물 삭제는 실행 취소로 복구 가능하니 걱정하지 마세요
